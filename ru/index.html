Быстрый, мощный и простой фрамеворк для блога

git clone https://github.com/lufficc/laravel-blog.git

  • Молниеносный

    Node.js даёт молниеносную скорость. Генерация сотни файлов занимает всего несколько секунд.

  • Поддержка Markdown

    Поддерживаются все возможности GitHub Flavored Markdown. Возможность использовать большинство плагинов Octopress в Hexo.

  • Размещение одной командой

    Нужна лишь одна команда для размещения сайта на GitHub Pages, Heroku и других хостингах.

  • Различные плагины

    Hexo обладает мощной системой плагинов. Можно установить различные расширения для Jade и CoffeeScript.
